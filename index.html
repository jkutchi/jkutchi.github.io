Pegboard Solitaire Solver (Python)

Given an initial state of a pegboard solitaire board, the solver will generate states until it has found a solution or no solution. If the solution is found the solution path is printed for the user. I implemented the breadth-first search, depth-first search, greedy-best search, and A* search algorithms to determine which state generated by my successor function to expand. CPU Scheduler Simulator (C++)

Given a randomly generated input of processes each with a process id, priority, CPU burst length, and arrival time, the scheduler will schedule the processes using the selected scheduling algorithm. I implemented first in first out, shortest burst first, and priority with preemption scheduling algorithms using queues and priority queues. After all the processes were finished, CPU utilization, throughput, average wait time, average turnaround time, and average response time were output for the user to compare the scheduling algorithms.

Producer-Consumer Simulator (C++)

Given an input of the number of producer threads, consumer threads, buffer size, and sleep time, the threads would produce or consume items from the buffer. Implemented the buffer as a circular queue. Used Windows library for producer and consumer threads, a mutex lock for accessing the buffer, and semaphores to keep track of how many full and empty slots are in the buffer.

CPU Temperatures Data Project (Python)

Given an input of CPU temperatures taken at thirty-second intervals, the program computed a linear interpolation between each of the temperatures and a least-squares linear regression over all the temperatures. Implemented without NumPy to learn how to create generalized functions for multiplying matrices, transposing matrices, and solving matrices using Gaussian elimination. Implemented later with NumPy to compare computational speedup.
